# Contributing to Our Project

Thank you for your interest int contributing! We're excited to see your ideas.
To ensure a smooth process, please follow these simple guidelines.

## How to Contribute

1. **Discuss First:** Before your start writing the code, please open an "Issue"
on GitHub to discuss the bug you want to fix or the feature you want to add. 
This helps us avoid duplicated work and make sure your idea aligns with
the project's goals.
